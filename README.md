# Svatební one‑page (minimal)

Krátký statický web pro svatbu Sylvy a Jiřího.

Preview locally:

1. Otevřete `index.html` ve vašem prohlížeči, nebo spusťte jednoduchý server v kořenovém adresáři:

```powershell
# z PowerShellu (Windows)
python -m http.server 8000
# pak otevřít http://localhost:8000/
```

Co je vytvořeno:
- `index.html` — plně obsahová stránka
- `css/styles.css` — styling a layout
- `js/main.js` — aktivní navigace a drobné JS

Poznámky:
- RSVP odkaz v `index.html` je placeholder: `https://example.com/rsvp` — nahraďte svým formulářem.
 - RSVP odkaz v `index.html` je: `https://forms.cloud.microsoft/e/8vtyNnnRJb` — nahraďte svým formulářem.
- Pokud chcete přesné písmo `Free Serif` a `Le Jour Script`, nahrajte webfonty (.woff/.woff2) a já je vložím.
